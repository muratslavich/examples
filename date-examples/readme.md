### Java Date formats

#### LocalDateTime

> Date and time without timezone

- `yyyy-MM-dd'T'HH:mm:ss` - "2004-10-19T10:23:54" - default Jackson ISO date format
- `yyyy-MM-dd HH:mm:ss` - "2004-10-19 10:23:54"

#### LocalDate
- `yyyy-MM-dd` - "2019-12-31" - default

#### LocalTime
- `HH:mm:ss` - "23:59:59" - default
